  • Flex

  • - Defines how a flex item will grow or shrink to fit the available space in container
    (it's actually a shorthand property for 3 other properties namely, flex-grow,shrink,basis)

  • Flex-Basis

  • - Sort of like width, but not. Specifies the ideal size of a flex item BEFORE it's placed into a flex container

    Flex basis also stated the initial ideal starting size of some flex items of a container
